This repository contains a complete HTML-only website created using **table layouts only** (`<table>`, `<tr>`, `<td>`).  
It includes multiple sections such as header, hero, features, about, traders, skills, pricing plans, articles, clients, contact, map, and footer.

This layout uses only **HTML tags + font tag + images**.  
No CSS, no JavaScript, no frameworks.
